# 桌面管理系统
# Desktop-Stytem
## 1.项目介绍
 * 桌面UI界面,程序可安装卸载运行
## 2.说明
*  桌面管理系统，实现对软件的管理运行
*  内置多个软件
*  软件交互
## 3.功能
  * __桌面UI__
  > 通过LVGL图形库来绘制页面,显示桌面软件图标，系统整体框架。
  * __顶部信息状态栏__
  > 显示时间，便签和状态
  * __后台运行状态栏__
  > 显示后台运行软件,可一键清理后台
   * __多功能按键__
  > 切换壁纸，安装软件，上传便签(调节亮度待开发)
## 4.技术
  >通用型链表  
  >文件IO
  >系统编程
  >网络编程
