# 桌面管理系统
# Desktop-Stytem
## 1.项目介绍
 * 桌面UI界面,程序可安装卸载运行
## 2.说明
*  桌面管理系统，实现对软件的管理运行
## 3.功能
  * __桌面UI__
  > 通过LVGL图形库来绘制页面,显示桌面软件
  * __状态栏__
  > 显示后台运行软件,可一键清理后台
   * __程序界面__
  > 单击软件运行,隐藏桌面，绘制软件界面,可通过home键返回桌面,并显示在状态栏中，单击可继续运行
## 4.技术
  >通用型链表  
  >文件IO
  >系统编程
